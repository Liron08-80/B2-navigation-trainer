BSC2 Navigation Data Flow Fault Trainer

Normal State vs Fault State · Data Source → DDU → Consumer System

Purpose

This trainer helps BSC2 learners identify what navigation data is missing, whether the problem is likely from the source, the DDU distribution path, or the consumer system.

Normal
Data is available and flowing.
Degraded
Data may be unreliable or partial.
Fault
Data is missing or not received.
DDU
Routes data. It does not create navigation data.

Normal Navigation Data Flow

All key navigation data is available and distributed through the DDU to the consumer systems.
